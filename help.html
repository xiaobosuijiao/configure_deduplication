工具简介

网络文本去重工具是一款专门用于处理网络设备配置文件的在线工具。它能够自动识别并删除重复的配置块，帮助网络工程师快速清理冗余配置，提高配置文件的可读性和维护性。

快速处理

支持大文件处理，能够快速处理数十万行的配置文件，提供实时进度显示。

智能识别

基于关键字自动识别配置块，支持自定义起始关键字和结束标记。

详细统计

提供详细的处理统计信息，包括处理时间、原始块数、保留块数和删除块数。

多种导出

支持导出处理结果和详细的重复块报告（JSON/CSV格式）。

快速开始

第一步：输入文本

您可以通过以下三种方式输入要处理的文本：

  1. 直接粘贴：在左侧输入框中直接粘贴您的配置文本
  2. 导入文件：点击"导入文件"按钮，选择要处理的配置文件（支持.txt、.conf、.cfg、.config、.log格式）
  3. 加载示例：点击"加载示例"按钮，使用内置的示例配置进行测试

第二步：配置参数

根据您的配置文件类型，调整以下参数：

  • 文本块起始关键字：配置块开始的关键字，如"controller"、"router"、"interface"等，多个关键字用英文逗号分隔
  • 块结束标记：配置块结束的标记，默认为"!"。如果为空，则自动检测到下一个关键字前结束

第三步：开始处理

点击"开始去重处理"按钮，工具将自动识别并删除重复的配置块。处理结果将显示在右侧输出框中。

第四步：导出结果

处理完成后，您可以选择：

  • 复制结果：将去重后的文本复制到剪贴板
  • 导出结果：将结果保存为文本文件
  • 导出详细结果：生成包含重复块详细信息的报告（JSON或CSV格式）

高级功能

配置块识别原理

工具通过以下规则识别配置块：

1. 扫描文本，查找以起始关键字开头的行
2. 从该行开始，收集后续所有行，直到：
- 遇到块结束标记（默认为"!"）
- 遇到下一个起始关键字
- 遇到纯数字行（如"1223"）
3. 将收集到的行作为一个完整的配置块
4. 比较所有配置块的内容，删除重复的块

示例配置

以下是一个典型的网络配置示例，包含重复的配置块：

controller flexe-group 1
bind controller flexe-200gi 0/1/0/1 phy-num 1
group-number 1
!
controller mtn-fgclient 1
bind mtn-client 1/301 fg-timeslot 0
fgclient-number 1
fg-oam
bas send enable
!
!
controller flexe-group 1 ← 这个块是重复的，将被删除
bind controller flexe-200gi 0/1/0/1 phy-num 1
group-number 1
!

使用技巧

  • 对于大型配置文件（超过10万行），工具会自动启用大文件处理模式，显示进度条
  • 您可以通过点击主题切换按钮在浅色和深色主题之间切换
  • 工具会记住您的主题偏好，下次访问时自动应用
  • 导出详细结果功能可以帮助您分析哪些配置块被重复使用了

注意事项

  • 工具处理的是文本级别的去重，不会解析配置的语义
  • 请确保起始关键字设置正确，否则可能无法正确识别配置块
  • 对于非常复杂的嵌套配置，建议先进行测试
  • 处理前建议备份原始配置文件

常见问题

Q1: 工具支持哪些文件格式？

支持.txt、.conf、.cfg、.config、.log等纯文本格式的文件。最大文件大小为100MB。

Q2: 如何处理包含嵌套配置的文件？

工具目前支持简单的嵌套配置识别。对于复杂的嵌套结构，建议适当调整块结束标记，或分多次处理。

Q3: 为什么有些相似的配置块没有被识别为重复？

工具进行的是精确匹配，只有内容完全相同的块才会被识别为重复。如果块中有微小差异（如空格、注释等），则不会被识别为重复。

Q4: 如何处理没有明显结束标记的配置？

可以将块结束标记留空，工具会自动检测到下一个起始关键字前结束。或者，可以设置一个在您的配置中不会出现的特殊字符作为结束标记。

Q5: 工具是否保存我的数据？

所有处理都在浏览器本地进行，数据不会上传到服务器。工具只会在本地存储您的主题偏好设置。

键盘快捷键

为了提高使用效率，工具支持以下键盘快捷键：

  • Ctrl + Enter：开始处理（当输入框获得焦点时）
  • Ctrl + C：复制处理结果（当输出框有内容时）
  • Ctrl + L：清空所有内容
  • Ctrl + E：加载示例
  • Ctrl + D：导出结果
  • Ctrl + H：打开帮助文档
  • Ctrl + T：切换主题
